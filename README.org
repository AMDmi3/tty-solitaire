* tty-solitaire
  ncurses-based klondlike solitaire game

** Run the game
#+BEGIN_SRC
$ git clone git://github.com/murilasso/tty-solitaire.git
$ cd tty-solitaire
$ make && bin/ttysolitaire
#+END_SRC

** Run the unit tests
#+BEGIN_SRC
$ make test
#+END_SRC
